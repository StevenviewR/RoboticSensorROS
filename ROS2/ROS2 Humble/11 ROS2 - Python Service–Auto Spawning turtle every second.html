  • Create a ROS Package with Python - Auto Spawning turtle every second at random position

Tips

ros2 run turtlesim turtlesim_node

  • ROS service:
    /spawn

  • ROS  service interface
    /turtlesim/srv/Spawn

  • Service requests:
    self.req.x = a
    self.req.y = b
    self.req.theta = 0.2

  • Random position:
    a = random.uniform(0, 10)
    b = random.uniform(0, 10)
